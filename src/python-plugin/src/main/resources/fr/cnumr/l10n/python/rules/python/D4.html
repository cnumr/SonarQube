Eviter d'utiliser des variables globales directement

Noncompliant Code Example

global_var = 'Hello Dolly';
def print_length() {
  print(len(global_var));
}
print_length();

Compliant Solution

global_var = 'Hello Dolly';
def print_length(local_var) {
  print(len(local_var));
}
print_length(global_var);
